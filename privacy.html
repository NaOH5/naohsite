Privacy Policy

This policy explains how NaOH Studios collects, uses, and protects user data when operating inside Discord servers.

1. Overview

By using NaOH Studios, you agree to this Privacy Policy. The bot is designed to improve server safety by detecting and automatically acting on flagged or malicious users using third-party security APIs.

2. Information We Collect

  • Discord User ID and username
  • Security risk data from Rotector and TASE APIs

3. How We Use Data

Data is strictly used only for moderation purposes.

4. Third-Party Services

NaOH Studios uses external services such as Discord API, Rotector API, and TASE API to evaluate user risk and detect malicious behavior. These services may process user-related identifiers to provide risk analysis.

5. Data Sharing

We do not sell or share personal data. Data is only shared with required APIs and Discord for functionality purposes.

6. Data Retention

Data is stored only as long as necessary for moderation and security purposes. If you appeal a ban caused by this bot, your Discord ID will be stored to ensure you won't be banned for the same reason again. The Hoster may request the deletion of the stored logs when applicable.

7. Security

We take reasonable measures to protect data from unauthorized access, alteration, or loss. However, no system is completely secure.

8. User Rights

You may request access, correction, or deletion of your data by contacting support.

9. Children’s Privacy

NaOH Studios products are not intended for users under Discord’s minimum age requirements. We do not knowingly collect data from minors or individuals that are below the age required for perspective countries. If you are one, please cease using our products.

10. Contact

For questions or requests, contact: naohstudios.legal@gmail.com
